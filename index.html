Color Identifier

Centered around interfaing with a color API, this is minimalistic web app that allows users to submit a hex code to the remote API. The user submitted information is checked for any potential errors (ie. the user enters an invalid hex code), then the app will use that data to dynamically change the information displayed on-screen, including the name and visual representation of that color.
